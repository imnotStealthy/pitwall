Overlay widgets

Checked widgets appear on screen over the game. Forza must run in Borderless mode.

Behaviour

Shortcuts: Ctrl+Alt+L lock/unlock · Ctrl+Alt+H hide/show.

Connection

In FH6: SETTINGS → Data Out → ON, IP 127.0.0.1, Port = your UDP port. Changing ports restarts the server and reloads the overlays — re-copy your OBS URLs from the OBS tab.

Size, opacity and which fields show — independently per widget. Changes preview live.
